## v0 目标

v0 阶段的目标是先跑通一个最小但完整的文件处理平台闭环：

1. 前端通过 Gateway 创建上传链接。
2. 文件直传对象存储。
3. Gateway 创建异步任务。
4. 任务进入队列。
5. Worker 消费任务。
6. 处理服务读取输入文件并生成结果。
7. 结果写回对象存储。
8. 前端查询任务并下载结果。

v0 不追求处理能力全面，而是优先验证架构主链路。
